## Quick Reference

**Task Types:**
- TWITTER_FOLLOW: Requires Twitter OAuth
- TWITTER_LIKE: Requires Twitter OAuth
- TWITTER_RETWEET: Requires Twitter OAuth
- TELEGRAM_JOIN: Requires Telegram verification
- REFERRAL: Automatic tracking
- CUSTOM: Manual or custom verification

**Point Ranges:**
- Easy (10-25): Simple actions
- Medium (30-60): Joining groups, watching content
- Hard (75-150): Creating content, code contributions
- Referral (100-10000): Based on referral count

**Usage:** Copy task details to Admin Panel → Tasks → Create New Task
